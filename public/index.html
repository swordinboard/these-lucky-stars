    Welcome!#

    Here you can find the free SRD content and related web apps for These Lucky Stars, a dynamic tabletop roleplaying system by Sword in Board Workshop.

    Get started with the Free SRD learn the system and start playing.

    Join the Reddit community and get involved! r/theseluckystars

    Current Status These Lucky Stars is in active playtesting. The rules presented here are stable and playable, but may change based on ongoing testing and feedback. Setting-specific content is still in development and kept within closed playtesting while the SRD and starter modules are in the process of being uploaded.

    Related Web Apps:#

    Digital Character Sheet This character sheet was made on a whim. It should have everything the basic paper sheet has. Check the READ ME at the top for more details.

    Attribute Converter This converter generates comparable stat arrays based on probability and overall strength. It can be used to transfer between TLS test versions, as well as the most common D20 systems. originally intended for a specific shift during playtesting, it serves as a handy tool for judging attribute arrays.

    Planet Name Generator This was made to generate planet names during the sci-fi playtesting. It really generates planet codes, not so much names.
